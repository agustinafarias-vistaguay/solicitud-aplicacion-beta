# Vistaguay Solicitudes - Planning Master

## Contexto
Este proyecto es una landing page para solicitudes de servicios de pulverización y siembra con drones de Vistaguay.

## Objetivos
- Proporcionar una interfaz intuitiva para agricultores.
- Validar datos de negocio, ubicación de lote y detalles técnicos de la aplicación.
- Enviar los datos a un webhook de n8n para su procesamiento.
